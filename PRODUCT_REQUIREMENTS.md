# Product Requirements - FastLane MVP

## Product Vision
A premium intermittent fasting companion that helps users start quickly, stay consistent, and understand progress with minimal friction.

## Primary Personas
1. Busy professional seeking weight and energy improvements.
2. Health-conscious beginner needing simple guidance.
3. Intermediate faster wanting consistency and trend tracking.

## MVP Features
- Conversion-first landing page.
- Onboarding preferences (goal, experience, protocol, schedule, reminders).
- Fasting timer (start/stop, protocol support, progress).
- Session history and streaks.
- Daily check-ins (energy, hunger, mood).
- Free vs Pro gating + contextual paywall.

## Non-Goals (MVP)
- Medical diagnosis or personalized medical plans.
- Deep nutrition logging.
- Full social/community features.

## Event Taxonomy
- landing_cta_clicked
- signup_started
- onboarding_completed
- first_fast_started
- first_fast_completed
- paywall_viewed
- trial_started
- subscription_started
- subscription_canceled
- weekly_active_user

## KPI Targets (Initial)
- Visitor -> signup: >= 12%
- Signup -> first fast start: >= 60%
- First fast start -> D7 retained: >= 35%
- Paywall view -> trial start: >= 8%
- Trial start -> paid conversion: >= 30%

## Compliance/Safety
- Prominent disclaimer: educational app, not medical advice.
- Encourage users with medical conditions to consult clinicians.
